# rp2040_example
an example of a configured project for the rp2040 microcontroller.

## how to use this?
you can use "make" to compile the firmware. the output is in the "build" directory.

use "make flash" to build and flash the firmware.

**attention!** when flashing the firmware for the first time, connect the board to the computer while holding down the "bootsel"/"boot" key.
